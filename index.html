Headings

As tags de headings são úteis para criar uma hierarquia visual na página. A tag <h1> é geralmente usada para o título principal da página, enquanto as tags <h2> até o <h6> são usadas para subtítulos ou seções menores.

Acessibilidade

Além de ajudar a organizar o conteúdo visualmente, as tags de headings são importantes para a acessibilidade. Leitores de tela usam as tags de headings para facilitar a navegação, permitindo que os usuários acessem diretamente as seções de interesse. Além disso, algumas tecnologias assistivas podem criar uma representação visual da estrutura da página, facilitando a experiência para usuários com deficiência visual.

SEO (Search Engine Optimization)

As tags de headings são cruciais para o SEO. Os motores de busca, como o Google, utilizam as tags de headings para entender o conteúdo e a estrutura da página, o que pode impactar diretamente na classificação nos resultados de pesquisa. Usar headings de forma consistente ajuda a destacar tópicos importantes e aumenta a visibilidade da página.

UX

Uma estrutura clara e organizada de headings melhora a experiência do usuário (UX). Páginas bem estruturadas são mais fáceis de navegar e entender, proporcionando uma leitura fluida. Isso pode aumentar o engajamento e direcionar os usuários para as informações mais importantes.

Boas práticas

Para usar as tags de headings corretamente, siga estas diretrizes:

  • Use as tags de headings em ordem: Comece com a tag <h1> para o título principal da página e siga com as tags <h2>, <h3> e assim por diante.
  • Não pule tags de headings: Usar tags fora de ordem pode confundir leitores de tela e motores de busca.
  • Use headings de forma significativa: Destaque títulos e seções importantes, evitando usá-las apenas para estilizar texto.

Parágrafos

A tag <p> é usada para criar parágrafos de texto. Cada parágrafo é exibido em um bloco único e se adapta automaticamente ao tamanho da janela do navegador ou da tela do dispositivo.

Os navegadores adicionam automaticamente uma quebra de linha antes e depois de um parágrafo, criando um espaço em branco que separa visualmente os parágrafos. Isso melhora a legibilidade, especialmente em textos longos.

A tag <p> também tem um papel semântico, indicando ao navegador e aos leitores de tela que o conteúdo dentro dela representa um parágrafo de texto. Para estilizar ou ajustar o espaçamento entre parágrafos, é recomendável usar CSS.

Formatação de texto

O HTML oferece várias tags para formatar o texto, como negrito, itálico, sublinhado e muito mais. Aqui estão algumas dessas tags:

  • negrito: a tag <b> é usada para deixar o texto em negrito, destaca o texto porém não o define como um texto importante.
  • forte: a tag <strong> é usada para destacar o texto como importante. Ela é geralmente exibida em negrito, mas o significado semântico é de maior importância do que o estilo visual.
  • itálico: a tag <i> é usada para deixar o texto em itálico. É geralmente usada para indicar um termo técnico, um pensamento ou uma palavra em outro idioma.
  • ênfase: a tag <em> é usada para enfatizar o texto. Ela é geralmente exibida em itálico, mas o significado semântico é de maior importância do que o estilo visual.
  • sublinhado: a tag <u> é usada para sublinhar o texto. É recomendado não utilizar essa tag em trechos que podem ser confundidos com links.
  • riscado: a tag <del> é usada para riscar o texto. Essa tag representa um texto que foi removido ou que não é mais válido.
  • sobrescrito: a tag <sup> é usada para colocar o texto em sobrescrito. O texto aparece acima da linha de base e é geralmente usado para indicar uma nota de rodapé ou uma referência. Como por exemplo, WWW[1].
  • subscrito: a tag <sub> é usada para colocar o texto em subscrito. O texto aparece abaixo da linha de base e é geralmente usado para indicar uma fórmula química como H2O.
  • destaque: a tag <mark> é usada para destacar o texto.
  • pequeno: a tag <small> é usada para diminuir o tamanho do texto. Como uma nota de rodapé ou uma legenda.

Listas

As listas são uma forma eficaz de organizar e apresentar informações em um texto. Em HTML, existem três tipos principais de listas que podem ser usadas:

  • listas ordenadas: a tag <ol> é usada para criar listas ordenadas, onde cada item é numerado. O caractere de numeração pode ser alterado usando o atributo type, alguns exemplos são: "1" (números), "A" (letras maiúsculas) e "I" (números romanos maiúsculos). Veja na prática:
    1. Item 1
    2. Item 2
    3. Item 3
  • listas não ordenadas: a tag <ul> é usada para criar listas não ordenadas, onde cada item é marcado com um ponto ou outro símbolo. O tipo de marcador pode ser alterado usando o atributo type, alguns exemplos são: "disc" (ponto preto), "circle" (círculo vazio) e "square" (quadrado). Veja na prática:
    • Item 1
    • Item 2
    • Item 3
  • listas de definição: a tag <dl> é usada para criar listas de definição, onde cada item é composto por um termo seguido de sua definição. Veja na prática:
    Termo
    - Definição

Estilos

Os estilos são uma parte importante do design de uma página web, permitindo que os desenvolvedores personalizem a aparência e o layout do conteúdo. No passado, era comum usar a tag <font> para definir o estilo do texto diretamente no HTML, como cor, tamanho e fonte. No entanto, com a evolução das boas práticas de desenvolvimento, essa tag foi descontinuada no HTML5.

Atualmente, os estilos são aplicados de maneira mais eficiente e flexível utilizando CSS (Cascading Style Sheets), que é uma linguagem de estilo separada do HTML. Em HTML, os estilos podem ser aplicados de várias maneiras, incluindo:

  • Estilos embutidos: os estilos embutidos são especificados diretamente no elemento HTML usando o atributo <style>. Por exemplo, você pode adicionar o seguinte código a um elemento p: <p style="color: red; font-size: 16px">

  • Estilos externos: os estilos externos são definidos em um arquivo CSS separado e vinculados à página usando a tag <link>, por exemplo, no <head> da página: <link rel="stylesheet" href="styles.css" /> Considerando que o arquivo styles.css contém o seguinte código: .externalStyle { color: green; font-size: 20px; }
